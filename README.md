# 2026_infection_transcript_usage
Github Repo for the paper "Single-cell atlas of transcript usage remodeling in antiviral immune responses across human populations""

Main developers: @rubenchazarra 

This repository contains the scripts used throghout the manuscript. The code is mostly in R.

    00_Processing: Code to run data processing
    00_utils: functions used through the scripts 
    01_Analysis: Code to re-run the analysis
    02_Figures: Code to create the figures
